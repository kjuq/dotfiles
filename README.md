# Overview

My ultimate dotfiles.

- OS: MacOS (Probably works on Linux)
- Terminal Emulator: Alacritty
- Terminal Multiplexer: Tmux
- Shell: Fish
- Editor: Neovim

# Usage

## Backup

`./main.sh b` backs up files listed in `targets.sh`

## Sync

`./main.sh s` creates symlinks
